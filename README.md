# Mithas — Bakery, Sweets & Cake House Website

A modern, responsive bakery website designed for **Mithas Bakery**.
The website showcases cakes, Indian sweets, bakery items, gallery, reviews, and allows users to **order directly through WhatsApp**.

This project demonstrates modern **frontend development using HTML, CSS, and JavaScript** with smooth animations, theme switching, and interactive UI components.

---

## 🌟 Features

* 🍰 Beautiful **Hero Section** with animations
* 🍪 **Interactive Menu Tabs** (Cakes, Sweets, Bakery)
* 🌙 **Dark / Light Theme Toggle**
* 💬 **WhatsApp Order Integration**
* 🖼️ **Gallery Section with Hover Effects**
* ⭐ **Customer Reviews Section**
* 📍 **Contact and Store Information Section**
* 🎯 **Smooth Scroll Animations**
* 🖱️ **Custom Cursor Effect**
* 📱 **Responsive Design**
